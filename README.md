# news-sentiment-analysis

## Dev Env Setup
### Create virtual env
```console
$ python3 -m venv venv
```
### Cctivate virtual env
```console
$ source venv/bin/activate
```
### Install dependencies
```console
$ pip install -r requirements.txt
```

### Setting the FLASK_APP environment variable
```console
$ export FLASK_APP=microblog.py
```

### run the app
```console
$ flask run
```
